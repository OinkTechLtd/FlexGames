# GoidaX

Браузерное расширение для разблокировки сайтов.

## Установка
1. Распакуй архив
2. chrome://extensions/
3. Режим разработчика → Загрузить распакованное расширение

## Функции
- Большая кнопка активации
- Таймер 60 минут
- Свой DNS
- Рекомендации зарубежных DNS

Создано как Goida-AI-Unlocker, но в виде расширения.
