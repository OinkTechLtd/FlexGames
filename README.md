# 🎮 Flex Games

Веб-приложение для каталога игр (Unity, HTML5 и другие) с автоматическим обновлением через GitHub Actions.

## 🚀 Возможности

- Отображение игр в виде сетки с картинками и категориями
- Поиск по названию и фильтр по категории
- Автоматическое обновление списка игр каждые 6 часов
- Лёгкая настройка источников для парсинга
