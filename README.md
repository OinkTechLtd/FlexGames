# GoidaX Browser Extension

Современное расширение для обхода блокировок в России и других странах.

## Функции
- Большая кнопка активации с таймером 60 минут
- Рекомендация зарубежных DNS (невозможно заблокировать)
- Сбор своего DNS для YouTube, TikTok, AI
- Автоматическое обновление списка DNS каждые 24 часа
- Приветствие при установке

## Установка
1. Скачай и распакуй проект
2. Открой `chrome://extensions/`
3. Включи "Режим разработчика"
4. Нажми "Загрузить распакованное расширение" и выбери папку

## Разработка
```bash
npm install
npm run update-dns
```

## GitHub Actions
Поисковой робот обновляет DNS автоматически.
